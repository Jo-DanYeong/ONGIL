# TMAP Android 지도 연동

온길 홈 지도는 공식 TMAP Android Vector Map SDK 3.7을 사용합니다.

## 설정

`local.properties`에 SK Open API 프로젝트에서 발급받은 앱 키를 저장합니다.

```properties
TMAP_APP_KEY=발급받은_앱_키
```

키는 Git에 커밋되지 않지만 Android `BuildConfig`를 통해 SDK에 전달되므로 APK에서 완전히 숨길 수 없습니다. SK Open API 콘솔에서 Android 패키지 `com.project.ongil`과 서명 인증서 제한을 설정해야 합니다.

## 현재 지도 구성

- 대치동 학원가 중심 좌표
- 학생·학부모 데모 마커
- A/B/C 픽업 후보 마커
- TMAP 실시간 교통 레이어
- 지도 SDK 생명주기(`onResume`, `onPause`, `onDestroy`) 연결
- 사용자가 직접 입력한 집·학원·학교 주소를 기기 로컬에 저장
- 저장 주소를 TMAP 주소 검색으로 좌표 변환하고 지도 마커로 표시
- 위치 권한 확인 후 현재 위치에서 선택된 픽업존까지 TMAP 보행자 경로 표시

픽업 후보 좌표는 현재 해커톤 시나리오 값입니다. 사용자의 현재 위치는 픽업존 도보 경로 버튼을 누르고 위치 권한을 승인했을 때만 읽습니다. 저장 장소와 현재 GPS는 온이 챗봇 요청이나 Gemini에 포함하지 않습니다.

## 사용 흐름

1. 홈 상단 `장소 설정`에서 집·학원·학교 주소를 저장합니다.
2. 픽업 가능 상황을 선택하면 온길이 픽업존을 추천합니다.
3. `픽업존 도보 경로`를 누르고 위치 권한을 승인합니다.
4. 현재 위치부터 추천 픽업존까지 TMAP 보행자 경로와 예상 거리·시간을 지도에 표시합니다.
